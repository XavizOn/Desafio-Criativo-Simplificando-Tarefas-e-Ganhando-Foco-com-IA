# Desafio Criativo: Simplificando Tarefas e Ganhando Foco com IA

## Passo 1 - Definindo a Intenção

### O que eu quero que a IA faça?

Gerar um plano de estudos para alguém que está iniciando o aprendizado na área de programação.

### Para quem?

Para estudantes iniciantes que não possuem experiência em programação.

### Qual o objetivo?

Aprender lógica de programação utilizando o Visualg.

### Resultado da Etapa

Quero que a IA gere um plano de estudos para iniciantes em programação, com o objetivo de aprender lógica de programação utilizando Visualg.

---

## Passo 2 - Adicionando Contexto e Restrições

### Contexto

O estudante está começando seus estudos em programação e precisa de uma orientação organizada para aprender os conceitos fundamentais da lógica de programação.

### Formato Desejado

O conteúdo deve ser apresentado em formato de cronograma semanal, organizado por semanas e contendo exemplos práticos para facilitar o aprendizado.

### O que deve ser evitado?

* Linguagem muito técnica.
* Conceitos avançados que possam dificultar o entendimento inicial.
* Assuntos que não estejam relacionados à lógica de programação básica.

### Resultado da Etapa

Considere o seguinte contexto: estudante iniciante sem experiência em programação. O conteúdo deve ter formato de cronograma semanal com exemplos práticos. Evite linguagem muito técnica e conteúdos avançados.

---

## Passo 3 - Construção do Prompt Final

Após definir a intenção, o contexto e as restrições, foi criado o seguinte prompt:

Quero que a IA gere um plano de estudos para iniciantes em programação, com o objetivo de aprender lógica de programação utilizando Visualg. Considere o seguinte contexto: estudante iniciante sem experiência em programação. O conteúdo deve ter formato de cronograma semanal com exemplos práticos. Evite linguagem muito técnica e conteúdos avançados.

---

## Explicação da Construção do Prompt

1. Gere um plano de estudo para alguém que está iniciando o aprendizado na área de programação.
2. O objetivo é aprender lógica de programação utilizando o Visualg.
3. Considere que o estudante não possui experiência prévia.
4. Organize o conteúdo em um cronograma semanal.
5. Inclua exemplos práticos para facilitar a aprendizagem.
6. Evite linguagem excessivamente técnica e conteúdos avançados.

Essas informações foram unificadas para formar um prompt claro, objetivo e capaz de gerar respostas mais precisas e adequadas ao público-alvo.
